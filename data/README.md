# Data folder

Place COCO2017 format data in this folder like this:  
```
data
├── COCO2017
│   ├── annotations
│	│	├── chair_sofa_train.json
│	│	└── chair_sofa_val.json
│   ├── chair_sofa_train
│   └── chair_sofa_val
└── README.md
```
Dataset in our project is Keypoint-5. 
Do preprocess the Anno file to specific .json format for cpn.
